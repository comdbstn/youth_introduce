# Tei Youth 명함 QR 랜딩페이지 - 기획서

## 프로젝트 개요

**목적**: 일본 비즈니스 네트워킹 및 명함 교환 행사에서 사용할 명함 QR코드와 연결되는 개인 브랜딩 웹사이트

**핵심 가치**: "나는 누구이며, 무엇을 만들어왔고, 어떤 가치를 제공할 수 있는가"를 한 페이지에 직관적으로 전달

**타겟**: 일본 비즈니스 파트너, 잠재 클라이언트, 네트워킹 참석자, 투자자, 협업 희망자

## 2. 기술 스택 및 구현

## 추천 기술 스택

`text프론트엔드: Next.js 14+ (React)
스타일링: TailwindCSS + Framer Motion
애니메이션: GSAP + ScrollTrigger
호스팅: Vercel (무료, 안정적, 빠른 배포)`

## 성능 최적화

- **모바일 우선 설계**: 일본 명함 교환 시 즉시 스캔하여 모바일로 접속하는 경우가 대부분
- **로딩 속도**: 3초 이내 First Contentful Paint 목표
    - Logo.png가 돌아가는 로딩 화면 필요
- **반응형 디자인**: 모든 디바이스에서 최적화된 경험 제공

## 5. 인터랙션 & 애니메이션

## 스크롤 애니메이션:[creativecorner+2](https://www.creativecorner.studio/blog/website-scroll-animations)

1. **Parallax**: 배경과 전경이 다른 속도로 움직임
2. **Fade-in on scroll**: 요소가 뷰포트에 들어오면 서서히 나타남
3. **Text reveal**: 텍스트가 한 줄씩 나타나는 효과
4. **Sticky sections**: 스크롤 시 섹션이 고정되어 컨텐츠가 변경됨

## 호버 효과:

- 프로젝트 카드: 이미지 확대 + GIF 재생
- 버튼: 그림자 효과 + 색상 변화
- 텍스트 링크: 밑줄 애니메이션

## 페이지 전환:

- "불빛 점멸 페이드": 섹션 간 전환 시 화면이 잠깐 어두워졌다가 밝아지는 효과

## 6. 모바일 최적화 전략

## 모바일 우선 설계 이유

- 일본 명함 교환 문화: 명함을 받자마자 즉시 QR 코드 스캔
- 대부분의 첫 방문자는 스마트폰 사용자
- 모바일 경험이 나쁘면 즉시 이탈

## 모바일 최적화 체크리스트:

- [ ]  모든 텍스트가 모바일에서 읽기 편함 (최소 16px)
- [ ]  터치 타겟 크기: 최소 44x44px[figma](https://www.figma.com/resource-library/mobile-first-design/)
- [ ]  이미지 최적화: WebP 포맷, lazy loading
- [ ]  스크롤 애니메이션: 모바일에서는 단순화 (과도한 효과 제거)[dribbble](https://dribbble.com/resources/career/product-designer-portfolio)
- [ ]  로딩 속도: 3G 환경에서도 5초 이내 로딩[framer](https://www.framer.com/blog/landing-page-best-practices/)
- [ ]  가로 모드 대응: 1280x720 해상도에서도 정상 작동

## 7. SEO & 접근성

## 기본 SEO 설정:

`xml<title>Tei Youth | Web Developer & Entrepreneur in Yokohama</title>
<meta name="description" content="Korean web developer and entrepreneur based in Yokohama. Specializing in React, Next.js, and full-stack development.">

## 접근성 (Accessibility):

- 시맨틱 HTML 사용 (`<header>`, `<main>`, `<section>`)
- 이미지에 alt 텍스트 추가
- 키보드 네비게이션 지원
- 색상 대비 비율: 최소 4.5:1 (WCAG AA 기준)[elementor](https://elementor.com/blog/best-portfolio-website-examples/)

## 8. 핵심 차별화 포인트

이 웹사이트가 단순한 명함 랜딩페이지가 아닌 이유:

1. **브랜딩 플랫폼**: 개인 브랜드 "Tei Souta"의 세계관을 시각화
2. **문화적 적합성**: 일본 비즈니스 문화(명함 교환, 신뢰 구축)에 최적화
3. **스토리텔링**: 한국 → 일본으로의 여정, 루틴 철학을 통한 차별화
4. **실용성**: 커피챗 예약, 포트폴리오 확인, 연락까지 한 페이지에서 완결
5. **기술력 과시**: 웹 개발자로서 최신 기술 스택과 디자인 감각을 증명
